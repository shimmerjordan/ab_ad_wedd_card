# 🌾 星露谷婚礼请帖 · Stardew Wedding Invitation

一份**可以玩的**婚礼电子请帖：星露谷(Stardew Valley)像素风小游戏，纯前端单文件，零构建零依赖。

宾客在游戏里操控新郎/新娘走动、跳跃、种菜、接水浇灌、施肥、钓鱼、交易，
逐步解锁请帖内容，最终在**婚礼殿堂**完成仪式、找到自己的喜宴桌位。

**钓鱼系统（星露谷式）**：金鲤鱼/银梭鱼/红鲷鱼按权重上钩、习性各异（红鲷爱乱窜）；
全程不脱钩 = **完美品相**（卖价 ×1.5）；拉扯中随机出现**喜礼盒**，罩住集满得金币/肥料/鱼饵/记忆碎片；
偶尔钓上水草破烂；等待期收竿不亏饵、任务鱼挣脱退饵。
**挖矿与博物馆捐赠**：镇西/湖东的**矿岩**用镐子开采（会重生），掉紫水晶/海蓝宝/黄水晶/红宝石/稀有钻石；
清路上的石头也有小概率掉宝石。把四大宝石捐给博物馆**捐赠柜**逐颗点亮，集齐触发「矿物学家」；宝石也能卖钱或送 TA。
**好感度 ❤（0-10）**：右上角木牌 HUD 显示婚礼倒计时/金币/心条；送礼（鸡蛋/鱼/向日葵/宝石，完美鱼与宝石反应更甜）、
彩蛋、金花收获都加好感；6 心 TA 回礼、10 心解锁「十心相印」；礼成自动圆满。
**经济系统**：钓鱼分鱼种计价、宝石收购、卖蛋/卖花 → 杂货店买种子/魔法肥料/渔具（真素材图标商店）。
**记忆碎片**：农作/钓鱼/彩蛋随机掉落关于新人的回忆（可配置）。
**世界细节**：农场狗「旺财」可摸头、湖面鸭子游弋、花田蝴蝶、面前有可互动物时头顶浮「A」气泡 + A 键发光、
金币变化飘字、殿堂内自动切换喜庆进堂曲 BGM。
**隐藏彩蛋/成就**：小鸡的祝福 / 鸡圈宝箱(跳篱笆) / 井底的愿望 / 草丛萤火虫 / 后巷小猫 / 旧靴子纸条 / 猫粮赞助商 / 十心相印 / 矿物学家。
**关于页彩蛋**：设置 → 关于 → 🎉。

## ⚙️ 自定义

### 基础信息
编辑 `js/config.js` 里的 `CONFIG`：姓名、日期、地点、电话、邀请函文案、流程、剧情分支台词。

### 婚礼回执 RSVP（统计来宾·零后端）
完整请帖里内置「📝 填写回执」卡片，用第三方问卷平台收集 **贵姓 / 来宾人数 / 祝福(选填)**，来宾免登录提交，你在平台后台看实时汇总并导出 Excel：
1. 去 **金数据 `jinshuju.net`** / **腾讯问卷 `wj.qq.com`** / **问卷星 `wjx.cn`** 免费建一个表单（建议字段：贵姓、来宾人数、祝福的话）；
2. 发布后复制表单的**分享链接**，粘贴到 `js/config.js → CONFIG.rsvp.url`；
3. 完成。`rsvp.embed=true` 时优先页面内嵌（iframe），平台禁止内嵌会自动改为新窗口打开；`url` 留空则该卡片显示配置提示、不影响来宾。

### DEBUG 模式（博物馆展品 / 桌位表 / 记忆碎片 / 任务直达）
1. 游戏内点 ⚙ 设置，**连点版本号 10 下**进入 DEBUG（版本号旁出现「退出 DEBUG」按钮）
2. 可视化编辑：🖼 博物馆展品（1-4 上墙挂画区，5-12 中央玻璃展柜）、🪑 桌位表（`姓名,桌号` 每行一位，桌号 1-15）、💫 记忆碎片
3. **任务直达**：DEBUG 面板有 Q0~Q5/礼成 按钮，一键跳到任意章节（自动补齐道具与 NPC 位置），殿堂大门也不再锁
4. 配置保存在本机浏览器；点 **⬇ 导出配置 JSON** 得到 `wedding-config.json`，与 `index.html` 一起部署宾客即可读到
5. 桌位编辑器为每位宾客生成**专属分享链接**（一键复制）

### 个人照片（博物馆展品/记忆碎片配图）
照片放进 **`assets/imgs/`**，配置里图片字段直接填**文件名**（如 `合照.jpg`）。
也支持完整 URL 或在 DEBUG 编辑器里上传转存。界面素材在 `assets/elem/`，两者互不干扰。

### 宾客桌位 GET 参数
- 自包含：`index.html?gn=张三&gt=3`（姓名+桌号直接进链接，无需配置文件）
- 查表式：`index.html?g=张三`（需部署 `wedding-config.json`）

宾客打开链接：标题屏显示欢迎横幅，殿堂内对应桌子金色高亮，最终请帖附桌位卡。

`?show=final|settings|fish|shop` · `?fphase=bite|reel&fsp=trout&ftr=1` 钓鱼阶段直达 · `?hearts=N` / `?gems=1` / `?donated=amethyst,ruby` 状态注入 ·

## 🛠 技术
Canvas 2D 像素渲染（spritesheet + 程序化绘制混合），Web Audio 实时合成 8-bit BGM 与音效，
多场景引擎（户外 + 2 内景），移动端虚拟摇杆，中文像素字体（缝合像素 via jsDelivr）。
素材图加载失败时自动回退到内置程序化像素画，离线/file:// 也能玩。
