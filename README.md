# 🌾 星露谷婚礼请帖 · Stardew Wedding Invitation

一份**可以玩的**婚礼电子请帖：星露谷(Stardew Valley)像素风小游戏，纯前端单文件，零构建零依赖。

宾客在游戏里操控新郎/新娘走动、跳跃、种菜、接水浇灌、施肥、钓鱼、交易，
逐步解锁请帖内容，最终在**婚礼殿堂**完成仪式、找到自己的喜宴桌位。

## 🎮 玩法主线

| 章节 | 玩法 | 解锁内容 |
|---|---|---|
| ① 花田初遇 | 找到 TA 对话（含三选一回忆分支） | 新人介绍 |
| ② 爱的浇灌 | 种星之种子 → 水井/湖边接水 → 浇水 →（可选施肥催熟金果）→ 收获×3 | 婚礼信息 + 倒计时 |
| ③ 同心鱼 | 湖边栈道钓鱼小游戏（按住A控制绿区） | 进入下一章 |
| ④ 查收喜讯 | 红色邮箱拆信 | 邀请函 |
| ⑤ 回忆博物馆 | 参观可配置的展品（图片+文字） | 进入终章 |
| ⑥ 婚礼殿堂 | 走上舞台举行仪式（誓言三选一），钢琴/香槟塔/蛋糕/礼花筒互动，按 GET 参数高亮**宾客桌位** | 当日流程 + 完整请帖 |

**经济系统**：钓鱼卖钱 → 杂货店买种子/魔法肥料。
**记忆碎片**：农作/钓鱼/彩蛋随机掉落关于新人的回忆（可配置）。
**6 个隐藏彩蛋**：小鸡的祝福 / 鸡圈宝箱(跳篱笆) / 井底的愿望 / 草丛萤火虫 / 后巷小猫 / 旧靴子纸条。
**关于页彩蛋**：设置 → 关于 → 🎉。

## ⚙️ 自定义

### 基础信息
编辑 `index.html` 顶部 `CONFIG`：姓名、日期、地点、电话、邀请函文案、流程、剧情分支台词。

### DEBUG 模式（博物馆展品 / 桌位表 / 记忆碎片）
1. 游戏内点 ⚙ 设置，**连点版本号 10 下**进入 DEBUG（版本号旁出现「退出 DEBUG」按钮）
2. 可视化编辑：🖼 博物馆展品（标题+文字+图片URL/本地图片）、🪑 桌位表（`姓名,桌号` 每行一位）、💫 记忆碎片
3. 配置保存在本机浏览器；点 **⬇ 导出配置 JSON** 得到 `wedding-config.json`，将其放到与 `index.html` 同目录一起部署，宾客即可读到
4. 桌位编辑器为每位宾客生成**专属分享链接**（一键复制）

### 宾客桌位 GET 参数
- 自包含：`index.html?gn=张三&gt=3`（姓名+桌号直接进链接，无需配置文件）
- 查表式：`index.html?g=张三`（需部署 `wedding-config.json`）

宾客打开链接：标题屏显示欢迎横幅，殿堂内对应桌子金色高亮，最终请帖附桌位卡。

## 🚀 部署
任意静态托管（GitHub Pages / Vercel / Netlify）：

```bash
# GitHub Pages：仓库 Settings → Pages → Deploy from branch → main
```

调试参数：`?auto=groom` 跳过选角 · `?at=x,y` 传送 · `?scene=museum|hall` · `?q=N` 跳任务 · `?show=final|settings|fish` · `?debug=1`

## 📁 文件
- `index.html` — 游戏版请帖（全部逻辑，单文件）
- `classic.html` — 经典滚动版（不想玩游戏的长辈也能看）
- `wedding-config.json` — （可选）DEBUG 导出的展品/桌位/碎片配置

## 🛠 技术
Canvas 2D 像素渲染（程序化地图/建筑/精灵），Web Audio 实时合成 8-bit BGM 与音效，
多场景引擎（户外 + 2 内景），移动端虚拟摇杆，中文像素字体（缝合像素 via jsDelivr）。
